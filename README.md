# ☄️ Asteroid Shooter Web

Classic Asteroids arcade game built with HTML5 Canvas and vanilla JavaScript.

## Features
- Rotating ship with thrust physics and screen wrapping
- 3-tier asteroid system (large → medium → small on split)
- Progressive difficulty — more asteroids per level
- Shield, Rapid-Fire and Triple-Shot power-ups
- Particle explosion effects on destruction
- Invincibility frames on respawn
- 3 lives system
- Level counter and high score (localStorage)
- Keyboard and virtual joystick (mobile)

## Controls
| Key | Action |
|-----|--------|
| ← → | Rotate ship |
| ↑ / W | Thrust |
| Space | Shoot |
| S / ↓ | Shield (when powered) |

## Tech Stack
- HTML5 Canvas
- Vanilla JavaScript (ES6+)
- Web Audio API
